## Второй кейс

### OCR Benchmark for Captcha-like Images

Исследование и сравнение способов подготовки изображений и OCR-моделей для коротких строк с сильными визуальными артефактами:

- собран benchmark на 32 размеченных примерах;
- протестированы варианты подавления фонового паттерна, blur, FFT, line removal, morphology и upscaling;
- проверены EasyOCR, Tesseract, OpenAI Vision, DeepSeek-OCR-2 и Chandra OCR 2;
- лучшим вариантом оказался `box_k5_soft_contrast + OpenAI Vision`;
- точность полного распознавания удалось поднять примерно с 5% до 40%.

Repository: https://github.com/ichgebegas/captcha-ocr-benchmark
